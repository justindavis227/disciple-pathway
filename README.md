# Disciple-Making Pathway

A single-page vision-cast site for Southeast Christian Church's disciple-making pathway,
plus a 4-Chair self-assessment and a fillable "My Circles" pocket card.

- `index.html` — the pathway (entry point)
- `inventory.html` — the 28-question 4-Chair assessment
- `my-circles.html` — fillable, printable 4×6 "My Circles" card
- `images/` — chair icons + assets

Static site — no build step. Deployed on Vercel.
